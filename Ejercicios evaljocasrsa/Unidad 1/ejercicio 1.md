


En el mundo del desarrollo web moderno, los lenguajes de marcas como HTML, CSS y SVG son esenciales para estructurar visualmente la información. SVG permite crear graficos directamente en el codigo, lo cual es util para elementos interactivos en la web

SVG funciona para crear imágenes usando etiquetas especiales dentro del código, en este caso se usaron las estiquetas circle y rect, que son para crear circulos y rectangulos respectivamente, luego se ponen las cordenadas en ambos, en el caso del circulo luego se escribe el radio y en el caso de el rectangulo el alto y el ancho y por ultimo, en ambos se Fill="color" para rellenarlo de un color en especifico 


A continuación se realiza el ejercicio propuesto:

```
<svg width="512" height="512" xmlns="http://www.w3.org/2000/svg">

<circle cx="300" cy="300" r="100" fill="#d36868"/>

<rect x="0" y="0" width="200" height="200" fill="#6c8acd"/>


</svg>

```


En resumen, el uso de SVG permite crear graficos personalizados de manera sensilla usando etiquetas de codigo. Gracias a etiquetas simples como ``<circle>`` y ``<Rect>``, es posible imprimir en pantalla formas básicas que ayudan con la interactividad y el diseño de las paginas web
